ClinicDesk System 
نظام إدارة عيادات متكامل ومؤمن تم تطويره باستخدام PHP و MySQL. يوفر النظام تجربة سلسة للأطباء، المرضى، ومديري النظام.

المميزات الرئيسية
نظام أدوار (Roles): (أدمن، طبيب، مريض) مع تحكم كامل في الصلاحيات.

إدارة المواعيد: حجز ذكي للمواعيد مع فحص للتضارب في الأوقات.

الوصفات الطبية: إمكانية رفع الملفات (PDF/Images) وتأمينها، مع نظام تنبيه ذكي للمريض.

أمان عالي: حماية ضد (SQL Injection, CSRF, XSS).

تقارير: إمكانية تصدير التقارير بصيغة CSV.

التقنيات المستخدمة
Backend: PHP (OOP)

Database: MySQL

Frontend: HTML5, CSS3, Bootstrap 4, JavaScript (Fetch API
